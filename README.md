#  Тестовое задание по направлению «Frontend-разработка»

## Приложение доступно на GitHub Pages:



## Инструкция по запуску приложения: 
1) Перейти в папку test-task-app и выполнить команду 

```bash
    npm install
```
2) В папке test-task-app для запуска Angular-приложения выполнить команду: 
```bash
    ng serve
```
или
```bash
    npm run start
```
3) В браузере открыть http://localhost:4200/ и начать пользоваться приложением.
